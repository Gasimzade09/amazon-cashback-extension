Amazon Cashback Extension

Automatically highlights best prices and tracks cashback on Amazon.

Features

  • Highlights the best prices on Amazon product pages.
  • Automatically applies your Amazon Affiliate Tracking ID.
  • Calculates cashback for users (5%) and owner (2%).
  • Works automatically, no popup or interface required.

Instructions

  1. Download or clone this repository.
  2. Open Chrome → Extensions → Developer mode → Load unpacked extension.
  3. Select the folder amazon-cashback.
  4. Done! Extension works automatically on Amazon product pages.
